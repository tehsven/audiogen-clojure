# audiogen

exploring audio generation with clojure

## deps
http://java.com/
http://brew.sh/

```
brew install git
brew install leiningen
```
```
java : 1.7.0_45
```

## install
```
git clone git@github.com:tehsven/audiogen-clojure.git
cd audiogen-clojure
lein deps
```

## usage
```
lein trampoline repl
```
