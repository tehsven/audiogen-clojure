# audiogen

exploring audio generation with clojure

## Dependencies
http://brew.sh/

brew install git

brew install clojure

brew install lein

## Installation
git clone git@github.com:tehsven/audiogen-clojure.git

cd audiogen-clojure

## Usage
lein run

; select the applet and start typing keys!
